# Котроль версії git
```
1.Скачати notepad++.
2.Скачати git.
3.Потрібно провести перші налаштування
Запускаємо cmd і вводимо команди
git config --global user.email "свою пошту"
git config --global user.email
git config --global user.name "login"
git config --global user.name
```

# Команди git
```
git status
git init
git status
git add .
git status
git commit -m"Simple view str"
```
